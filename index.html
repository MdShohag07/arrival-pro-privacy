Privacy Policy — Arrival Pro

Chrome extension "Arrival Pro". Last updated: 10 July 2026.

Arrival Pro ("the extension") is a workflow tool that helps authorised users process product arrivals on Miinto's partner portal (proxy-arrival.miinto.net). This policy explains exactly what data the extension handles and why.

Data we handle

  • Google account email address. Read once through Chrome's identity.email API for a single purpose: to check whether the signed-in user is on the approved-users list (access control). If the optional one-device lock is enabled by the account owner, the email is sent only to a private server the account owner deploys and controls. The email is never sold, shared with third parties, or used for advertising.
  • Page content on proxy-arrival.miinto.net. Style IDs, sizes, brand, and category are read from the page solely to perform the automation (filling sizes, collecting Style IDs). This data is stored locally in the browser using chrome.storage.local.
  • Local usage counts. The number of products submitted or rejected, and the Style IDs of submitted products, are stored locally to power the extension's own dashboard.

What we do not do

  • We do not collect, transmit, or store personal data on our own servers.
  • We do not use analytics, tracking, or advertising of any kind.
  • We do not sell or transfer user data to third parties.
  • We do not use or transfer user data for purposes unrelated to the extension's single purpose.
  • We do not use or transfer user data to determine creditworthiness or for lending purposes.

Where data is stored

All data stays on the user's own device, inside the browser's local extension storage. The only exception is the optional one-device lock check, which contacts a server that the account owner deploys and controls.

Data retention and deletion

Data is stored locally and can be cleared at any time using the Reset and Clear buttons inside the extension, or by removing the extension from Chrome. Removing the extension deletes all of its locally stored data.

Permissions

Each permission the extension requests is used only to deliver the features described on its Chrome Web Store listing: reading the account email for access control, reading and filling the Miinto arrival pages, opening and stepping through tabs, and storing settings and counts locally. No permission is used for tracking.

Children's privacy

Arrival Pro is a workplace tool intended for authorised adult users and is not directed to children.

Changes to this policy

If this policy changes, the updated version will be published on this page with a new date.
